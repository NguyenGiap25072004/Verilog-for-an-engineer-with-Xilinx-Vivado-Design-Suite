Q ) If i assigned any number(may be octal hexadecimal decimal), it will be store internally in binary only in verilog language? (Arvind Kumar)

A ) yes reg is datatype used to store unsigned binary data (single bit or multibit) while integer is used to specify signed 32-bit data. Then while initializing or assigning a value you can use any number format e.g

reg [3:0] a = 12; -- Number without format specifier will be integer so it will be as if you are

reg [3:0] a = 0000 0000 0000 0000 0000 0000 0000 1100 ; so since you declare reg size as 4-bit it will consider LSB 4-bits.

you can use any number format to initialize or assign data to variable e.g.

reg [3:0] a = 4'h c ; -- Number format Hexadecimal

reg [3:0] a = 4'b 1100; -- Number format Binary

reg [3:0] a = 4'o 14 ; --- Number format octal

So reg can only store binary number whereas any number format can be used to initialize or assignment.
