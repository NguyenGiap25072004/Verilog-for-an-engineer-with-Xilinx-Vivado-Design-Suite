What is the difference between ' reg [3:0] a ' and ' reg a[3:0] '  (Arvind Kumar)

Answer :

A) Packed Array : When you declare size of the array before variable name (reg [3:0] a), it is termed as Packed array in SystemVerilog.  The packed array represent continuous set of bits or data.

module tb;
  reg [3:0]a = 4;
  
initial begin
  $display("The Value %0d",a);
end
  
endmodule


B)  Unpacked array: When you declare dimension after the variable name, it is termed as Unpacked array in SystemVerilog. It is way of specifying collection of the distinct elements.

Example with Unpacked Array :

Valid Operation :

module tb;
  reg a [3:0]= {1,1,1,0};
  
initial begin
  $display("The Value %0d",a[3]);
  $display("The Value %0d",a[2]);
  $display("The Value %0d",a[1]);
  $display("The Value %0d",a[0]); 
end
endmodule

Invalid Operation:

module tb;
  reg a [3:0]= {1,1,1,0};
  
initial begin
  $display("The Value %0d",a);
end
  
endmodule
