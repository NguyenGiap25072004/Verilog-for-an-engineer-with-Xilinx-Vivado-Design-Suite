1) After 'Migrate to RTL' and a constraints file is generated, how to get to view Package and Design view? (Kambadur)

Sol : Complete your Source Code -> Click on Run Synthesis -> Allow Synthesis to complete -> Select Open Synthesis Design from the Synthesis Completed Pop-out -> This will load Device and Package View

If you  won't See device and package view -> Go to Layout -> Choose I/O Planning View.
